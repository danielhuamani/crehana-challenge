# Crehana challenge

## Description

Crehana's todo list project for the challenge

## Requirements

requirements to execute the project:

- `docker`

## Project installation
===============

1. Clone this repository
2. cd docker
2. cp .env_template .env
3. Execute command: make build

## Run
===============
- `make up`.


## Uni Test
===============
-  Open new terminal
- run `make docker-test`.
